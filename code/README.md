# Instruções para compilação e execução do sistema

>[!NOTE]
> Incluir todas as informações necessárias para compilar e executar o programa com sucesso.

## Compilação e Execução

### Pré-requisitos

- Qt Creator 18.0.0 (versão em que o projeto foi compilado e testado)
- Qt 6.10.0 (versão em que o projeto foi compilado e testado)
- CMake 3.16.0 (versão em que o projeto foi compilado e testado)

### Compilação

Para compilar o projeto, você deve:

- Clone este repositório
- Instale o software Qt Creator na versão 6.10.0
- Abra o Qt Creator e abra o projeto selecionando o arquivo code/CMakeLists.txt
- Configure o projeto ao abrir, selecionando a versão Desktop para Debug e Release, depois clique em "Configure Project"
- Na barra lateral esquerda, clique em "Build" para compilar o projeto

### Execução

Para executar o projeto, você deve:
- Na barra lateral esquerda, clique em "Run" para executar o projeto
- O projeto será executado e a janela do jogo será exibida
